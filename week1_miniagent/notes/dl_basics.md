# 深度学习笔记

## 优化器对比

### SGD
随机梯度下降，每次用一个小批量计算梯度。优点：简单、泛化性好。缺点：收敛慢，容易在鞍点震荡。

### Adam
结合了动量法和RMSprop。自适应学习率，对稀疏梯度效果好。默认学习率0.001。
公式：m_t = β1 * m_{t-1} + (1-β1) * g_t，v_t = β2 * v_{t-1} + (1-β2) * g_t^2

### AdamW
Adam + 权重衰减解耦。在Transformer模型中常用。比Adam泛化性更好。

## 激活函数

### ReLU
公式：f(x) = max(0, x)。优点：计算简单，缓解梯度消失。缺点：神经元死亡问题（负梯度永远为0）。

### GELU
高斯误差线性单元，Transformer默认激活函数。公式：x * Φ(x)，其中Φ是标准正态分布的累积分布函数。

### SiLU / Swish
f(x) = x * sigmoid(x)。在深层模型上优于ReLU。
